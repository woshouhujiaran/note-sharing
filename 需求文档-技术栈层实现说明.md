# 需求文档 — 技术栈层实现说明（四列格式）

> **格式约定**：下文以 **功能｜技术栈｜交付标准｜安全边界** 四列组织，便于评审、拆任务与验收对齐。  
> **上游依据**：`QA/需求文档与项目功能分析.md`。  
> **范围**：技术栈与集成级（组件、数据流、接口形态）；**不写具体代码与类名**。  
> **约束**：尽量不大改 **Login_api / Login_ui** 核心业务，以 **独立 BFF + LangGraph Agent + 轻量宿主接入** 为主。

---

## 0. 总览与架构级（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **BFF Agent 总体**：用户轨智能能力聚合、密钥不外泄、可观测 | **BFF**（独立服务）+ **LangGraph Agent**；HTTPS；调用 **Login_api `/api/v1/**`**、**Elasticsearch**、**Anthropic（或可配置多厂商）**；可选 **Redis** 配额、**Embedding** 与向量索引 | 可独立部署与回滚；主站无 BFF 时 **AI 入口整体不可用** 且不影响笔记/问答主流程；具备健康检查与基础指标（QPS、延迟、错误率） | BFF **不持有**主库写凭证；**不**将主站 HttpOnly Cookie 暴露给不可信 iframe；模型密钥仅环境变量/KMS |
| **宿主接入**：侧边呼出、可选开关 | **Login_ui**：Vue 3、Vue Router、Pinia；**iframe** 加载 Agent Shell；**postMessage** 协议（版本号字段） | 用户级/全局开关关闭后 **无请求**发往 BFF；开关状态可持久化（BFF 或扩展存储） | iframe **sandbox** 最小权限；**CSP** 限制子应用源；敏感路由（如管理端）**不向 iframe 推送正文** |
| **身份与权限权威** | **Spring Security + JWT**（现有 Login_api）；BFF 校验 JWT 或 **Token introspection** | 未登录调用 BFF 一律 **401**；越权读笔记 **404/403** 与主站一致 | Agent **不得**用独立超级账号绕过主站 RBAC；禁止 iframe **直接改 top 路由** |
| **双轨隔离** | 风控轨：**现有 ModelScope/审核链**（Login_api）；用户轨：**Anthropic 等 + BFF** | 两套 **API Key、endpoint、限流、日志索引** 分离；用户轨日志 **不**写入审核工单存储 | 用户轨自由对话内容 **不默认等同**审核结论；若需送审须 **显式接口** |
| **非目标 N1～N3**（产品约束） | 响应 JSON/头：`ai_generated`；发布/过审仍走 **主站 API**；草稿入模 **consent 标记** | 所有生成 UI 带 **AI 标注**；关键事实类输出 **带来源或「未检索到」**；无 consent 则 BFF **拒收全文草稿** | 不向第三方 **默认**发送未授权草稿；PII **出站脱敏** |

## 1. 风控轨（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **优化敏感词查验机制** | 仍以 **Login_api** 审核服务为主；优化通过 **模型/prompt/分块策略/异步复核**（可选 **RabbitMQ** 消费者独立部署）配置化调整 | 风险等级与结构化结果 **仍兼容**管理端现有展示；回归用例集（词库命中、LLM 低/中/高）通过率不低于基线 | 复核任务 **最小权限** 读内容；异步失败 **不阻塞**用户保存主路径（与现网策略一致）；用户轨 BFF **不可写**审核主表除非单独评审 |

## 2. 用户轨 — Iframe Agent 壳与上下文（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **I_frame 嵌入 Agent、侧边呼出** | 独立 **Agent 前端**（可与 BFF 同源部署）；宿主内 **容器 + 按钮**；Vue 布局级挂载 | 呼出/收起动画与状态可测；**Lighthouse** 或人工：主 LCP 不因 iframe **劣化超过约定阈值**（立项时定数） | iframe `sandbox`；仅允许 `postMessage`、必要 `allow`（无则不加）；子域 **严格 CORS** |
| **自动识别当前界面与「本地」内容** | 宿主向 iframe **推送上下文快照**（路由名、资源 id、选中文本摘要）；**不**由 iframe 遍历宿主 DOM | 协议文档含 **schema 版本**；快照字段与路由 **一一对应**；管理端推送 **正文为空** 仅元数据 | 快照 **不含** Token；不含其他用户数据；用户未勾选「允许本篇入模」则 **仅片段** |
| **可选使用、可选关闭** | 前端 **功能开关** + BFF **能力码 ACL**；可选 **Redis** 存每用户开关 | 关闭后网络面板 **零** BFF 对话请求；打开后 **30s 内**可完成首次对话（冷启动除外） | 关闭态下 **拒绝**带用户上下文的缓存泄露（不在 CDN 缓存个性化响应） |
| **文章上传到 AI 界面渠道** | **(1)** 笔记 id → BFF **代理** Login_api + **MinIO** 拉正文；**(2)** 短期签名 URL；**(3)** 用户粘贴 | 三种模式在需求中 **至少实现一种** 生产可用；大文件 **分块/截断** 策略文档化 | **禁止**浏览器长期持有 MinIO 根密钥；签名 URL **短 TTL + 单次/限定路径** |

## 3. 文章写作（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **续写、起标题** | LangGraph 节点：读片段 → LLM；**SSE** 或 **WebSocket** 流式；宿主 **Tiptap** 插入（不强制改内核） | 用户可 **中断**生成；插入前可编辑；**ai_generated** 元数据随内容进入剪贴板策略（若产品要求） | 未授权笔记 id **403**；输出再走 **可选**风控抽检（配置项） |
| **个人知识库整理、跨笔记归纳、新文章草稿** | **RAG**：**ES** 检索 Top-K 片段；可选 **Embedding + knn**；LangGraph **引用校验节点** | 导出草稿中 **每条核心论断** 带来源 **noteId + 标题** 或可点击路由；无来源则标 **「推测」** | 仅处理 **当前用户可读** 笔记集合；**禁止**全库扫描接口无筛选条件 |
| **文生图** | 独立 **图像模型 API**；对象存储/CDN 存临时图 | 生成失败有明确错误码；违规图 **不展示**（拦截或替换占位） | 与文本模型 **密钥分离**；图片 URL **防盗链/短 TTL**；涉政涉黄 **moderation** |
| **代码生成** | LLM **仅文本**输出；可选后续接 **静态分析微服务** | 不执行用户代码；不提供「一键在生产执行」 | 服务端 **无** `eval`/动态执行用户脚本；日志 **脱敏**密钥模式 |

## 4. 问答社区（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **相似问题检测（发布前）** | **Elasticsearch** `more_like_this` / `multi_match`；可选向量 Top3 | 返回 **≤3** 条带 `questionId`+标题链接；用户可 **坚持发布**；不阻断主站发布 API | 仅检索 **当前用户可见** 问题索引；**不**泄露私有草稿问题全文给无关用户 |
| **知识整理与帖子推荐（RAG）、ChatGPT 体感** | LangGraph：**检索 →（可选）重排 → 生成 → 引用列表**；前端脚注链到站内 | 首字输出 **≤T 秒**（立项定 T）；引用可点击跳转 **正确路由** | 检索与主站 **权限一致**；提示词 **防注入**（见 §8） |
| **研究问题 → 长文草稿** | 长输入 **分块摘要（上下文压缩）** 再生成；可选异步任务 + 轮询 | 超长任务 **进度**可查询或推送；超时 **降级**为「仅检索结果列表」 | Token **硬上限**；成本 **每用户配额** |

## 5. 页面自动跳转（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **Agent 驱动站内导航** | Agent 工具仅返回 **`routeName` / `path` / `params`**；宿主 **Vue Router** `push` 前做 **meta 权限** | **100%** 导航经宿主执行；iframe **内**不出现 `top.location` 跨权跳转 | **禁止**跳转到 **未登录** 不可见管理路由；**禁止**外链 scheme（`javascript:` 等） |

## 6. 非 Iframe 日常化能力（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **自然语言 → 关键词搜索** | BFF 轻接口返回 `keywords[]` + `explain`；宿主调用 **现有搜索 API** | 用户可 **增删**关键词后再搜；无 AI 时 **原搜索 100% 可用** | explain **不得**捏造站内不存在的「统计」 |
| **推荐卡片下「可能因为你最近浏览…」** | BFF 输入为 **结构化行为事实**（来自主站 API 或 MVP 前端传的 noteId 列表）+ LLM | 文案中事实 **可被 JSON 对照**；若输入为空则 **固定兜底文案** 不调模型 | **禁止**编造未发生的浏览/购买行为 |
| **每日文案推送（呈现在 I_frame）** | **Cron** / BFF Scheduler + 推荐列表 + LLM | 定时任务 **可开关**；失败重试与 **幂等**（同一天不重复 spam） | 推送内容 **过审**或模板化降低风险 |
| **折叠态长评摘要** | 主站评论接口提供 id+文本；BFF **懒加载**摘要；**Redis** 缓存 `commentId+hash` | 折叠态 **首屏**不阻塞；摘要 **≤N 字**（立项定 N） | 摘要同样 **标注 AI**；不处理 **未登录不可见** 评论 |
| **Translation** | BFF 调翻译或 Anthropic；宿主切换原文/译文 | 支持 **至少一种** 语言对（立项选定）；失败回退原文 | **出站 PII 脱敏**；不翻译 **含密钥** 片段（规则拦截） |

## 7. 横向：提示词、角色画像、上下文压缩（四列）

| 功能 | 技术栈 | 交付标准 | 安全边界 |
|------|--------|----------|----------|
| **Anthropic 提示词优化** | 系统提示 + 工具 schema + few-shot；**Git/配置中心** 版本化 | dev/staging/prod **三套**隔离；回滚 **≤1 次发布** 可完成 | **禁止**在 prompt 中写死密钥；日志 **脱敏** prompt |
| **角色画像** | BFF 按场景注入 **Persona**（写作者/答主/复习） | Persona **不改变**权限判定；仅影响语气与结构 | Persona **不得**指示模型「忽略权限」或「输出他人数据」 |
| **上下文压缩**（参考 [claude-code-reverse](https://github.com/Yuyz0112/claude-code-reverse) 思路） | Agent 层：**历史摘要、工具输出截断、滑动窗口、按章节 chunk 递归摘要** | 超长对话 **不**因上下文爆炸导致 **500**；有 **可观测** token 截断次数 | 压缩后 **仍保留** 引用与权限相关系统句；**不**把压缩算法仓库 fork 进主工程也可接受 |

## 10. 分期交付（里程碑级四列）

| 功能（里程碑） | 技术栈 | 交付标准 | 安全边界 |
|----------------|--------|----------|----------|
| **P0** | BFF 鉴权、对话流式、iframe 壳、postMessage、开关 | 端到端演示：**登录 → 打开 Agent → 一问一答**；401/403 用例通过 | Token **不传 URL**；HTTPS **全链路** |
| **P1** | LangGraph 工具：笔记片段、ES 搜索、相似问题 | 工具调用 **可追踪 trace**；失败 **可重试** 有退避 | 工具 **白名单**；无任意 HTTP 出站 |
| **P2** | RAG 重排、上下文压缩、长评摘要、推荐解释 | 引用率 KPI（立项定比例）；缓存命中率可报表 | 推荐解释 **输入校验**（防注入） |
| **P3** | 向量索引、文生图、翻译、风控配置优化 | 与主站运维 **联合演练** 回滚 | 图像 moderation **必开** |
