# 项目需求文档（改）

## 基础结构目的

BFF Agent —— BFF + LangGraph Agent

## 功能需求

### 非功能需求

| 编号 | 非目标                      | 说明                                                         |
| ---- | --------------------------- | ------------------------------------------------------------ |
| N1   | 「AI 一定正确」             | 所有生成内容需标注 AI 生成、鼓励用户校对；关键事实场景要求引用站内来源。 |
| N2   | 全自动代发帖 / 默认自动过审 | 发布、通过审核仍由用户或人工流程主导；AI 不自动替代决策。    |
| N3   | 未经明示同意处理高敏感数据  | 不向第三方默认发送未发布草稿全文；需单独授权或脱敏策略。     |

### 功能需求

#### 风控轨

- 优化敏感词查验机制

#### 用户轨

- I_frame 嵌入Agent 侧边呼出 自动识别当前界面本地内容和功能 
- 可选使用，可选关闭

##### 详细功能构想

- 文章写作功能
  - 文章续写、起标题等基础功能
  - 个人知识库内容整理（多篇学习、跨笔记主题归纳、自动生成新文章等）
    - 注意：这里需要提供文章上传到AI界面的渠道/模式
  - 文生图
  - 代码生成
- 问答社区功能
  - 相似问题检测
    **发布前**：ES 或向量检索 Top3 相似问题链接；用户可跳转合并或坚持发布。
  - 知识整理与帖子推荐（RAG）
    提出问题后经过知识库整理和构想获得结论，给出引用文章的链接，体感同ChatGPT。
    研究问题 → Top-K 片段检索 → 带 **笔记 ID 引用** 的长文草稿。
- 页面自动跳转
  - 这里需要注意规范权限
- 非I_frame功能（即日常化功能）
  - 关键词搜索——自然语言表述搜索时通过Agent抽离关键词，等效关键词查询。关键词可由个人再作加入或删除。
  - 在推荐卡片下展示「可能因为你最近浏览了…」；结合真实行为日志防瞎编。
  - 每日文案推送（呈现在I_frame）
  - 问答社区折叠态长评摘要生成
  - Translation
